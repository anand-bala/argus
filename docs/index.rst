Installing
----------

.. code-block:: bash

   pip install argus-temporal-logic

.. note::
   The name is `argus-temporal-logic` as opposed to just `argus` or `pyargus` as there
   are multiple similarly named projects on PyPI. The current package name minimizes
   ambiguity.
